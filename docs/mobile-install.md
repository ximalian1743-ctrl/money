# 在手机上安装和使用

部署完成后（例如可以访问 `https://money.ximalian.cc.cd`），就可以在手机上安装为 PWA，之后和原生 app 使用感受接近：有独立图标、全屏运行、无浏览器地址栏。

## iOS（iPhone / iPad）

iOS 目前只有 **Safari** 能真正安装 PWA，其它浏览器（Chrome / Edge iOS 版）都是内核壳，不行。

1. 用 **Safari** 打开站点
2. 点底部工具栏中间的「分享」按钮（方框向上箭头）
3. 下滑菜单，找到「**添加到主屏幕**」
4. 名称默认是「账本」，可以改，点「添加」
5. 主屏幕出现一个绿色 ¥ 图标，点进去就是账本

⚠️ iOS 限制：

- **Safari 清除网站数据会丢 IndexedDB**。定期从设置里导出 JSON 备份
- **iOS 长期不用（7 天以上）可能清理 PWA 数据**。不要依赖 PWA 作为唯一存储
- 不支持 Web Push，不会有通知

## Android

推荐 **Chrome**（Edge / Firefox 也行）。

1. 用 Chrome 打开站点
2. Chrome 通常会在底部弹出「添加到主屏幕」横幅；点它即可
3. 没弹也没关系：点右上角菜单「⋮」→「**添加到主屏幕**」或「安装应用」
4. 确认名称，点「安装」
5. 启动器里会出现应用图标

Android 的 PWA 体验和原生 app 非常接近，存储也不会被系统轻易清除。

## 首次使用

1. 打开 app，默认有 7 个账户（邮储存折、现金、中行储蓄、微信、西瓜卡、PayPay 信用卡…）
2. 去「**设置**」：
   - 填 AI Endpoint（兼容 OpenAI 的地址，例如 `https://xxx/v1/chat/completions`）
   - 填 AI Key（⚠️ 不要填付费 key）
   - 选模型，点「加载模型列表」可以拉取
   - 填 CNY ↔ JPY 汇率
3. 去「**账户**」：把每个账户的「初始余额」改成真实数额，保存
4. 回到总览，应该看到正确的总资产

## 日常记账

**文字录入**：
- 「AI 录入」→ 粘贴或输入「今天午饭 38 块，微信」
- AI 解析后给一个 draft，确认无误点保存

**拍小票**：
- 「AI 录入」→ 切到图片标签 → 拍照或从相册选小票
- AI 读出金额、商家、日期，确认后保存

**手动录入**：
- 「手动录入」填表（支出/收入/转账/还款/余额调整）

**查账**：
- 「总览」看资产和账户分布
- 「账本」翻所有交易，可软删除（数据库里保留 `deletedAt`）

## 数据备份（重要）

「设置 → 数据备份」三个按钮：

- **导出 JSON**：把当前所有账户、交易、设置打包下载，文件名带日期
- **导入 JSON**：用之前的备份恢复（会覆盖当前数据）
- **清空重置**：清库 + 重新种默认账户（危险）

**建议节奏**：每周一次导出到网盘 / 邮件给自己。手机换机、重装 PWA、清浏览器数据都要靠这个恢复。

## 更新版本

代码 push 到 main → Cloudflare 自动重建 → 下次打开 PWA 时 Service Worker 检测新版本、自动 reload 到最新。不需要手动操作。

## 遇到问题

- **打开白屏 / 卡旧版本**：iOS 在「设置 → Safari → 高级 → 网站数据」找到站点并删除；Android 在系统「应用信息 → 存储 → 清除数据」。然后重新打开站点，重装 PWA。**注意先导出备份。**
- **AI 不响应**：去「设置」确认 endpoint 和 key 还有效；打开浏览器 devtools 看是不是 CORS / 401。
- **数字对不上**：去「账户」页检查初始余额，或在「账本」翻历史交易有没有重复或类型选错。
